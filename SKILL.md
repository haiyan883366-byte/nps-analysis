# NPS 分析与转化报告生成 v3

根据教育行业 NPS 问卷 Excel 底表，按 **辅导老师×组长** 维度 + **组长** 维度双表输出 NPS 指标和转化意向报告。
Excel 带 **ColorScale 色阶**（绿→白→红渐变），一眼识别高低分。

## 使用场景

- 按辅导老师分组计算 NPS
- 同时看组长维度的聚合表现
- EXCEL 带自动色阶标注，汇报即用
- 输出带颜色标注的 HTML 报告

## 输出内容

### Excel（一个文件，3 个 sheet）

| Sheet | 内容 | 色阶 |
|-------|------|------|
| **辅导维度** | 每位辅导老师的 NPS / 推荐 / 贬损 / 报名 / 考虑 / 拒绝 | ✅ 有 |
| **组长维度** | 每位组长的聚合 NPS | ✅ 有 |
| **汇总对比** | 两个维度放同一页，上下排列 | ✅ 各自独立色阶 |

### HTML

上下两段：辅导表 + 组长表，各自带背景色标注。

## NPS 计算口径（v3）

**底表数据为 1-11 分制**（问卷系统存储的值包含 1-11，0 分不存在）：

| 分类 | 底表分值 | 标准含义 | 
|------|----------|----------|
| **推荐者** | 10-11 分 | 对应标准 0-10 量表的 9-10 |
| 中立者 | 8-9 分 | 对应标准 0-10 量表的 7-8 |
| **贬损者** | 1-7 分 | 对应标准 0-10 量表的 0-6 |

- **NPS** = 推荐者占比 − 贬损者占比

### ⚠️ 数据校验

脚本会自动检查 `nps`/`推荐值` 列是否只包含 **1-11** 的值：
- 如果存在 1-11 之外的值 → **打印告警**，提示用户检查底表
- 异常行会被**自动剔除**，仅用 1-11 数据计算

## 数据要求

Excel 底表必须包含以下列（**支持多种列名格式**，脚本自动检测并适配）：

| 必需列候选名 | 说明 | 取值 |
|-------------|------|------|
| `nps` / `推荐值` / `【推荐值】` | NPS 推荐值 | **1-11 整数** |
| `是否报名` / `是否考虑报名长期班` | 转化意向 | 1=报名, 2=考虑, 3=拒绝 |
| `年级` / `grade_names` | 所在年级 | 如"一年级" |
| `辅导` / `counselor_name` / `辅导姓名` | 辅导老师姓名 | 文本 |
| `组长` / `组长姓名` | 所属组长 | 文本 |

其他列（user_id、主讲、喜爱度等）为可选，不影响计算。

## 执行流程

### 1. 确认参数

- `input_file` — Excel 底表路径（必填，可由用户 @文件 传入）
- `grade` — 筛选年级（如"一年级"，可选，默认不分年级算全量）
- `min_samples` — 最小样本量阈值，默认 10
- `output_dir` — 输出目录（可选，默认输出到 input_file 所在目录）

### 2. 运行计算脚本

```bash
python <skill-dir>/scripts/nps_calculator.py "<input_file>" --grade "<grade>" [--min-samples <num>] [--output-dir "<dir>"]
```

脚本自动完成：
1. 读取 Excel + 列名适配
2. **数据校验**：检查 nps 列是否在 1-11，超出范围告警
3. 辅导维度分组计算 + 过滤小样本
4. 组长维度聚合计算
5. 输出 Excel（3 sheet、ColorScale 色阶）
6. 输出 HTML（双维度、带颜色）

## 边界情况处理

### 小样本年级

当年级样本量少且辅导老师分散（单老师样本 < min_samples），脚本输出可能很少。

此时应**自动降级**：
1. 组长级汇总（正常出）
2. 辅导级降低 min_samples 到 5 或 2
3. 明确告知用户数据局限性

### 数据异常

当 nps 列出现 1-11 之外的数值时：
1. 打印详细告警（异常数量、分布）
2. 自动剔除异常行
3. 提示用户检查底表是否正确

## 示例

```
用户："帮我分析桌面上的 NPS问卷表.xlsx，只看一年级的"
→ python scripts/nps_calculator.py "C:/.../NPS问卷表.xlsx" --grade "一年级"
→ 输出：一年级NPS分析表.xlsx（3 sheet）+ 一年级NPS分析表.html
```
