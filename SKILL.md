# NPS 分析与转化报告生成 v4（统一版）

**合并了原 `nps-analysis` 和 `nps-merge-analysis` 两个 skill，一个脚本覆盖所有场景。**

## 两种使用模式

### 模式一：双底表（推荐）

同时提供《NPS转化底表》和《行课底表》：

```
NPS转化底表 + 行课底表
  → Phase 0: 列名重命名 → 三重键匹配教学组 → 填充组长姓名
  → Phase 1: 年级筛选 → NPS计算（辅导+组长维度+色阶）→ HTML
  → 产出：合并后底表 + NPS汇总表 + HTML报告
```

### 模式二：单底表（兼容）

只提供《NPS转化底表》（底表已含组长列）：

```
NPS转化底表
  → Phase 1: 年级筛选 → NPS计算（辅导+组长维度+色阶）→ HTML
  → 产出：NPS汇总表 + HTML报告
```

## 输出内容

| 文件 | 双底表 | 单底表 | 说明 |
|------|--------|--------|------|
| **{年级}合并后底表.xlsx** | ✅ | — | 两表合并后的完整底表（含教学组） |
| **{年级}NPS分析表.xlsx** | ✅ | ✅ | 3 sheet（辅导维度/组长维度/汇总对比），ColorScale 色阶 |
| **{年级}NPS分析表.html** | ✅ | ✅ | 双维度 + 颜色标注 |

## 两表合并逻辑（Phase 0）

### 数据要求

**NPS转化底表** 需含以下列（自动适配列名变体）：
- 推荐值：`nps` / `q1` / `推荐值` / `【推荐值】` — **1-11 整数**
- 转化意向：`q10` / `是否报名` / `是否考虑报名长期班` — 自动重命名为 `是否考虑报名长期班`
- 匹配用列：`counselor_name`（辅导）、`grade_names`（年级）、`fk_course`（课程ID）

**行课底表** 需含以下列（自动检测 sheet 和列名）：
- `辅导名` — 辅导老师姓名
- `年级` — 年级
- `课程ID` — 课程ID
- `教学组` — 教学组/组长名称
- 可选：`组长姓名` — 组长名称（无此列时用教学组代替）

### 匹配策略

采用 **三重键 → 双键降级** 策略：

1. **三重键匹配**：(辅导名, 年级, 课程ID) — 精确匹配，解决辅导重名问题
2. **双键降级**：(辅导名, 年级) — 三重键未命中时使用
3. 匹配到的教学组填入 `组长姓名` 列

### 自动检测

- **Sheet 检测**：自动遍历所有 sheet，找到包含所需列的那个
- **列名变体**：`counselor_name` / `辅导` / `辅导名` / `辅导姓名` 等自动适配
- **编码兼容**：支持 GBK/UTF-8 编码的列名

## NPS 计算口径

**底表数据为 1-11 分制**（问卷系统存储的值包含 1-11）：

| 分类 | 底表分值 | 标准含义 |
|------|----------|----------|
| **推荐者** | 10-11 分 | 对应标准 0-10 量表的 9-10 |
| 中立者 | 8-9 分 | 对应标准 0-10 量表的 7-8 |
| **贬损者** | 1-7 分 | 对应标准 0-10 量表的 0-6 |

- **NPS** = 推荐者占比 − 贬损者占比

### ⚠️ 数据校验

脚本会自动检查 NPS 推荐值是否只包含 **1-11** 的值：
- 如果存在 1-11 之外的值 → **打印详细告警**，提示用户检查底表
- 异常行会被**自动剔除**，仅用 1-11 数据计算

## 色阶规则

| 列 | 方向 | 高值色 | 低值色 |
|----|------|--------|--------|
| NPS | 正向 | 🟢 绿 | 🔴 红 |
| 推荐率% | 正向 | 🔵 蓝 | 浅蓝 |
| 贬损率% | **反向** | 🔴 红 | 🟢 绿 |
| 报名率% | 正向 | 🟢 绿 | 🟠 橙 |
| 拒绝率% | **反向** | 🔴 红 | 🟢 绿 |
| 考虑率% | 中性 | — | — |

## 执行流程

### 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| `--nps-file` | ✅ 是 | NPS转化底表路径 |
| `--xingke-file` | 否 | 行课底表路径（提供则走双底表模式） |
| `--grade` | 否 | 筛选年级，如"一年级" |
| `--min-samples` | 否 | 最小样本量阈值，默认 10 |
| `--output-dir` | 否 | 输出目录，默认同 nps-file |

### 命令行调用

```bash
# 双底表模式（推荐）
python scripts/nps_calculator.py \
  --nps-file "NPS转化底表.xlsx" \
  --xingke-file "行课底表.xlsx" \
  --grade "一年级"

# 单底表模式（兼容）
python scripts/nps_calculator.py \
  --nps-file "NPS问卷表.xlsx" \
  --grade "一年级"
```

## 边界情况处理

### 小样本年级
1. 组长级汇总正常输出
2. 辅导级降低 min_samples 到 5 或 2
3. 明确告知用户数据局限性

### 数据异常
当 nps 列出现 1-11 之外的数值时：
1. 打印详细告警（异常数量、分布）
2. 自动剔除异常行
3. 提示用户检查底表

### 低匹配率
如果教学组匹配率 < 80%：
1. 列出未匹配的辅导老师名单
2. 提醒检查行课底表完整性
