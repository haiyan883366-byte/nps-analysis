# NPS 分析与转化报告生成 v2

根据教育行业 NPS 问卷 Excel 底表，按 **辅导老师×组长** 维度 + **组长** 维度双表输出 NPS 指标和转化意向报告。
Excel 带 **ColorScale 色阶**（绿→白→红渐变），一眼识别高低分。

## 使用场景

- 按辅导老师分组计算 NPS
- 同时看组长维度的聚合表现
- EXCEL 带自动色阶标注，汇报即用
- 输出带颜色标注的 HTML 报告

## 输出内容

### Excel（一个文件，3 个 sheet）

| Sheet | 内容 | 色阶 |
|-------|------|------|
| **辅导维度** | 每位辅导老师的 NPS / 推荐 / 贬损 / 报名 / 考虑 / 拒绝 | ✅ 有 |
| **组长维度** | 每位组长的聚合 NPS | ✅ 有 |
| **汇总对比** | 两个维度放同一页，上下排列 | ✅ 各自独立色阶 |

### HTML

上下两段：辅导表 + 组长表，各自带背景色标注。

## 数据要求

Excel 底表必须包含以下列（**支持三种列名格式**，脚本自动检测并适配）：

| 【收获感】 | 学习收获感 | 1-5 整数 |
| 是否考虑报名长期班 | 转化意向 | 1=报名, 2=考虑, 3=拒绝 |
| 年级 | 所在年级 | 如"一年级" |
| 辅导 | 辅导老师姓名 | 文本 |
| 组长 | 所属组长 | 文本 |

### 格式 B（夏1期）

| 列名 | 说明 | 取值 |
|------|------|------|
| 推荐值 | NPS 推荐值 | 1-11 整数 |
| 喜爱度 | 课程喜爱度 | 1-5 整数 |
| 收获感 | 学习收获感 | 1-5 整数 |

| grade_names | 所在年级 | 如"一年级" |

| 组长 | 所属组长 | 文本 |

### 格式 C（夏2期）

| 列名 | 说明 | 取值 |
|------|------|------|
| nps | NPS 推荐值 | 1-11 整数 |
| q2喜爱度 | 课程喜爱度 | 1-5 整数 |

## 执行流程

### 1. 确认参数

- `input_file` — Excel 底表路径（必填，可由用户 @文件 传入）
- `grade` — 筛选年级（如"一年级"，可选，默认不分年级算全量）
- `min_samples` — 最小样本量阈值，默认 10
- `output_dir` — 输出目录（可选，默认输出到 input_file 所在目录）

### 2. 运行计算脚本

```bash
python <skill-dir>/scripts/nps_calculator.py "<input_file>" --grade "<grade>" [--min-samples <num>] [--output-dir "<dir>"]
```

脚本自动完成：
1. 读取 Excel + 列名适配
2. 辅导维度分组计算 + 过滤小样本
3. 组长维度聚合计算
4. 输出 Excel（3 sheet、ColorScale 色阶）
5. 输出 HTML（双维度、带颜色）

### 3. 色阶规则

| 列 | 方向 | 高值色 | 低值色 |
|----|------|--------|--------|
| NPS | 正向 | 🟢 绿 | 🔴 红 |
| 推荐率% | 正向 | 🔵 蓝 | 浅蓝 |
| 贬损率% | **反向** | 🔴 红 | 🟢 绿 |
| 报名率% | 正向 | 🟢 绿 | 🟠 橙 |
| 拒绝率% | **反向** | 🔴 红 | 🟢 绿 |
| 考虑率% | 中性 | — | — |

### 4. NPS 计算口径

- **NPS** = 推荐者占比 − 贬损者占比
- **推荐者**：推荐值 ≥ 9 分的用户
- **贬损者**：推荐值 ≤ 6 分的用户
- **被动者**：推荐值 = 7 或 8 分的用户（不计入 NPS 公式但包含在样本中）

### 5. 转化意向计算口径

- **报名**：是否考虑报名长期班 / 是否报名 = 1 的占比
- **考虑**：= 2 的占比
- **拒绝**：= 3 的占比

## 边界情况处理

### 小样本年级

当年级样本量少且辅导老师分散（单老师样本 < min_samples），脚本输出可能很少。

此时应**自动降级**：
1. 组长级汇总（正常出）
2. 辅导级降低 min_samples 到 5 或 2
3. 明确告知用户数据局限性

## 示例

```
用户："帮我分析桌面上的 2026年7月NPS问卷表.xlsx，只看一年级的"
→ python scripts/nps_calculator.py "C:/.../2026年7月NPS问卷表.xlsx" --grade "一年级"
→ 输出：一年级NPS分析表.xlsx（3 sheet）+ 一年级NPS分析表.html
```
