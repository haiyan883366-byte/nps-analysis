# NPS 分析与转化报告生成

根据教育行业 NPS 问卷 Excel 底表，按辅导老师×组长维度生成 NPS 指标和转化意向报告。

## 使用场景

当用户提供一份 NPS 问卷 Excel 文件并要求：
- 按辅导老师分组计算 NPS
- 列出推荐者占比、贬损者占比
- 附带报名/考虑/拒绝比例
- 输出带颜色标注的表格报告

## 数据要求

Excel 底表必须包含以下列（列名需完全匹配）：

| 列名 | 说明 | 取值 |
|------|------|------|
| 【推荐值】 | NPS 推荐值 | 0-10 整数 |
| 【喜爱度】 | 课程喜爱度 | 1-5 整数 |
| 【收获感】 | 学习收获感 | 1-5 整数 |
| 是否考虑报名长期班 | 转化意向 | 1=报名, 2=考虑, 3=拒绝 |
| 年级 | 所在年级 | 如"一年级" |
| 辅导 | 辅导老师姓名 | 文本 |
| 组长 | 所属组长 | 文本 |

其他列（用户序号、主讲等）为可选，不影响计算。

## 执行流程

### 1. 确认参数

从用户输入中提取以下参数，缺失时询问：

- `input_file` — Excel 底表路径（必填，可由用户 @文件 传入）
- `grade` — 筛选年级（如"一年级"，可选，默认不分年级算全量）
- `min_samples` — 最小样本量阈值，默认 10（样本量低于此值的辅导老师不展示）
- `output_dir` — 输出目录（可选，默认输出到 input_file 所在目录）

### 2. 运行计算脚本

使用 Python 执行核心脚本：

```bash
python <skill-dir>/scripts/nps_calculator.py "<input_file>" --grade "<grade>" [--min-samples <num>] [--output-dir "<dir>"]
```

如果用户未指定 grade，省略 `--grade` 参数。

脚本将自动：
1. 读取 Excel 底表
2. 按指定年级筛选（如有）
3. 按「辅导」×「组长」分组
4. 计算 NPS、推荐、贬损、要报、考虑、不报指标
5. 过滤样本量 < min_samples 的组
6. 按组长分组、组内 NPS 降序排列
7. 输出 `<grade>NPS分析表.xlsx` 和 `<grade>NPS分析表.html`

### 3. 展示结果

执行完成后，使用数据预览或 files 交付生成的两个文件给用户。

在回复中提供：
- 数据概览（总样本数、有效辅导老师数、组长分布）
- Top 3 & Bottom 3 的 NPS 和要报情况
- 关键发现（异常值、亮点等）

### 4. NPS 计算口径

- **NPS** = 推荐者占比 − 贬损者占比
- **推荐者**：推荐值 ≥ 9 分的用户
- **贬损者**：推荐值 ≤ 6 分的用户
- **被动者**：推荐值 = 7 或 8 分的用户（不计入 NPS 公式但包含在样本中）

### 5. 转化意向计算口径

- **要报**：是否考虑报名长期班 = 1 的占比
- **考虑**：是否考虑报名长期班 = 2 的占比
- **不报**：是否考虑报名长期班 = 3 的占比
- 三者之和 ≈ 100%（可能存在舍入误差）

## 输出格式

### Excel 文件

| 辅导 | 组长 | 样本量 | NPS | 推荐 | 贬损 | 要报 | 考虑 | 不报 |
|------|------|--------|-----|------|------|------|------|------|

数值均以百分比形式展示，保留 1 位小数。

### HTML 文件

带颜色标注的可视化表格，配色规则：
- **NPS 列**：绿色渐变（值越高颜色越深）
- **推荐列**：蓝色渐变
- **贬损列**：蓝色渐变（反直觉：值越高越需关注，蓝色越深）
- **要报列**：绿红渐变（高值为绿，低值为红）
- **考虑列**：无底色
- **不报列**：蓝色渐变

## 示例

用户：帮我分析桌面上的 "2026年7月NPS问卷表.xlsx"，只看一年级的

执行：
```bash
python scripts/nps_calculator.py "C:/Users/xxx/Desktop/2026年7月NPS问卷表.xlsx" --grade "一年级"
```

输出：`一年级NPS分析表.xlsx` + `一年级NPS分析表.html`

## 边界情况处理

### 小样本年级（如四年级仅 100 人 / 58 位辅导老师）

当年级样本量少且辅导老师分散（单老师样本 < min_samples），脚本输出可能为空（0 位辅导老师）。

此时应**自动降级**为两步策略：
1. **组长级汇总**：直接按组长分组计算，过滤样本量 < 3 的组长
2. **辅导级（低阈值）**：辅导老师维度输出 min_samples=2 版本，标注「样本极小，仅供参考」

两种输出均用临时 Python 脚本生成（非核心脚本），输出文件命名为 `{grade}NPS分析表_组长级.xlsx/html` 和 `{grade}NPS分析表_辅导级.xlsx/html`。

在回复中明确告知用户数据局限性，建议以组长级为主。
